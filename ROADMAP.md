# Project "X" Roadmap

This roadmap describes the development plan for Project "X" Roommate Bot MVP.

## Current Vision

Project "X" aims to become a student-focused housing and roommate matching platform.

The first product is a Telegram MVP that helps students create profiles, find compatible roommates, communicate safely through mutual match and reduce uncertainty in housing search.

## Current MVP

Implemented:

- Telegram bot
- profile creation
- profile photo
- moderation
- moderation comments
- roommate search
- saved preferences
- compatibility score
- compatibility explanation
- favorites
- reports
- verified profiles
- verification requests
- mutual match
- hidden profiles
- contact feedback
- admin panel
- analytics
- broadcast
- MVP report
- language switch
