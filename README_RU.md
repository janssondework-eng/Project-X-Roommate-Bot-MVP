# Project X — MVP Telegram-бота для поиска соседей

## О проекте

Project X Roommate Bot — это MVP Telegram-бота для подбора соседей и поиска жилья среди студентов.

Проект разработан как промежуточная версия Project X — платформы для поиска жилья и соседей, ориентированной на студентов, переезжающих в другой город для учёбы.

## Цель проекта

Основная цель MVP — проверить ключевую продуктовую гипотезу:

> Студентам нужен простой, безопасный и структурированный способ поиска совместимых соседей до аренды жилья.

MVP позволяет протестировать пользовательские сценарии, структуру анкет, процессы модерации и базовую логику подбора пользователей до разработки полноценной платформы.

## Проблема

Студенты, переезжающие в другой город, часто сталкиваются со следующими трудностями:

- сложно найти надёжного соседа;
- недостаточно информации о привычках и образе жизни потенциального соседа;
- поиск жилья и соседей разбросан по чатам, социальным сетям и неструктурированным группам;
- существуют риски при общении с незнакомыми людьми;
- отсутствует механизм подбора пользователей по совместимости.

## Решение

Бот предоставляет структурированный Telegram-интерфейс, через который пользователь может:

- создать анкету;
- указать предпочтения по жилью и образу жизни;
- загрузить фотографию;
- отправить анкету на модерацию;
- искать потенциально подходящих соседей;
- получать совпадения по выбранным критериям.

## Функциональность MVP

### Возможности пользователя

- Регистрация пользователя
- Создание анкеты
- Редактирование анкеты
- Загрузка фотографии
- Поиск соседей
- Базовая система matching
- Постоянное меню навигации

### Возможности администратора

- Модерация анкет
- Одобрение анкет
- Отклонение анкет с комментариями
- Управление пользовательским контентом

### Работа с данными

- Хранение данных в SQLite
- Хранение анкет пользователей
- Хранение результатов подбора
- Хранение статусов модерации

## Продуктовая аналитика

Проект изначально проектировался с учётом дальнейшего внедрения продуктовой аналитики.

Потенциальные метрики:

- количество зарегистрированных пользователей;
- количество созданных анкет;
- процент заполнения анкет;
- процент одобренных анкет;
- конверсия в match;
- количество возвратов пользователей;
- конверсия приглашений друзей;
- удержание пользователей (Retention);
- причины отклонения анкет.
