Project “X” — Roommate Bot MVP
English Version
Overview

Project “X” Roommate Bot is an MVP Telegram bot designed to help students find compatible roommates and housing options through structured profiles, basic matching logic and moderation.

The project was created as an intermediate MVP version of Project “X”, a student housing platform concept focused on making the process of finding roommates safer, simpler and more structured.

Problem

Students moving to a new city often face several problems:

difficulty finding reliable roommates;
lack of structured information about lifestyle habits and housing preferences;
safety concerns when contacting strangers;
fragmented housing search across chats, social media and informal groups;
no simple matching system based on compatibility criteria.
Solution

The MVP provides a Telegram-based interface where users can:

create a personal profile;
describe lifestyle and housing preferences;
upload a photo;
search for potentially compatible roommates;
submit profiles for moderation;
receive basic matches based on selected criteria;
interact through a more structured and safer process.

Product Scope

This MVP was focused on testing the core product hypothesis:

Students need a simpler, safer and more structured way to find compatible roommates and housing options.

The MVP helps validate:

profile structure;
questionnaire logic;
moderation flow;
basic matching mechanics;
early user experience;
potential product analytics metrics.
Analytics Perspective

The project can be further developed with product analytics metrics such as:

number of registered users;
number of created profiles;
profile completion rate;
profile approval rate;
match conversion rate;
returning users;
rejected profile reasons;
invite conversion rate;
