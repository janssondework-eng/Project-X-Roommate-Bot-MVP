# User Flow

## User Journey

### Step 1 — Registration

User starts the Telegram bot and creates an account.

↓

### Step 2 — Profile Creation

User fills in:

- Name
- Age
- University
- Budget
- Lifestyle preferences
- Housing preferences

↓

### Step 3 — Photo Upload

User uploads profile photo.

↓

### Step 4 — Moderation

Profile is submitted for review.

Admin can:

- Approve profile
- Reject profile
- Send correction comments

↓

### Step 5 — Search

User starts roommate search.

↓

### Step 6 — Matching

System identifies compatible profiles.

↓

### Step 7 — Interaction

Users can review matches and continue communication.

## Future Flow

Future versions may include:

- Mutual match confirmation
- Housing listings
- In-app communication
- Analytics dashboard
