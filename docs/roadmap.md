# Roadmap

## MVP Stage (Completed / In Progress)

- User registration
- Profile creation
- Profile editing
- Photo upload
- Basic matching logic
- SQLite database
- Admin moderation
- Invite-a-friend system

## Phase 2

- Improved matching algorithm
- English language support
- Better profile filtering
- Housing search module

## Phase 3

- Mutual match confirmation
- Advanced recommendation system
- Analytics dashboard
- User retention tracking

## Phase 4

- Full web platform
- Mobile application
- Integrated housing marketplace
- AI-assisted matching

## Long-Term Vision

Create the leading student housing and roommate matching ecosystem for students moving to new cities.
