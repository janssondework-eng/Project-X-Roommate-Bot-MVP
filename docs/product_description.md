# Product Description

## Project Overview

Project X is a student housing and roommate matching platform designed to simplify the process of finding compatible roommates.

The first implementation of the idea is a Telegram bot MVP that allows students to create profiles, specify housing preferences and search for potential roommates through a structured process.

## Target Audience

- Students moving to another city for education
- International students
- First-year university students
- Students looking for shared accommodation

## Problem Statement

Finding a compatible roommate is often difficult because:

- Information is fragmented across chats and social media
- Users know little about each other's lifestyle
- There is no structured matching process
- Safety and trust issues exist

## Proposed Solution

Project X provides:

- Structured user profiles
- Preference-based matching
- Moderation system
- Safe communication workflow
- Future housing marketplace integration

## Value Proposition

Project X helps students:

- Find compatible roommates faster
- Reduce risks associated with shared housing
- Save time during housing search
- Improve transparency between future roommates
