# Changelog

All notable changes to Project "X" Roommate Bot MVP are documented in this file.

## v4.6 — Mutual Match, Moderation Comments and Language Switch

### Added

- Mutual match system.
- Users can respond to profiles instead of directly opening contact.
- Profile owner can accept or reject a match request.
- Contact opens only after mutual interest.
- Moderation comments.
- Admin can request changes instead of only approving or rejecting a profile.
- User receives a clear explanation of what should be fixed.
- Language switch.
- Added Russian / English language selection.
- Basic multilingual user flow support.

### Improved

- Safer communication between users.
- More respectful contact process.
- Better moderation workflow.
- Better support for future international users.
